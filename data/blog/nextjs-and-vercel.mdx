The blog is written in Next.JS using the Tailwind Next.JS starter git ()). Site gets a near perfect Lighthouse score with no effort.

It was remarkably easy to setup and deploy to Vercel (). The custom domain setup including SSL was also very straightforward. Whole thing took 30 minutes starting
from scratch with no experience of Vercel.

CD is setup automatically (and I mean automatically) from Github and triggered when you push commits to master.
